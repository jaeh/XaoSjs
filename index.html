    Try It

    This is a real-time fractal zoomer. Using it is simple: left-click to zoom in, right-click to zoom out.

    Your browser doesn't seem to support the <canvas> tag. Try Chrome or Firefox.

    About

    XaoS.js is a real-time fractal zoomer. Instead of drawing a box and waiting for the fractal to re-draw, it allows you to dive into the fractal in realtime.

    It a JavaScript adaptation of the Hubicka algorithm from XaoS (http://xaos.sf.net). The zooming code was adapted from Java by way of JAME (http://fractalwalk.net).
